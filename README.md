2. Prepare: Azure RBAC, FortiGate image/licensing, subnets/UDRs/NSGs.
3. Deploy: follow [DEPLOYMENT_GUIDE](./docs/DEPLOYMENT_GUIDE.md).
4. Validate & operate: see [RUNBOOK](./docs/OPERATIONAL_RUNBOOK.md).
